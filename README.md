# HomeFixr – Home Services Booking Platform

HomeFixr is a full-stack web application that connects customers with local service workers such as electricians, plumbers, carpenters, cleaners, and more. The platform allows users to browse workers, book home services, and manage service requests through a modern and responsive interface.

## Features

* Worker registration system
* Service booking functionality
* View available workers
* Booking management
* REST API using Flask
* MySQL database integration
* Responsive frontend using HTML, CSS, and JavaScript

## Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Python
* Flask
* Flask-CORS

### Database

* MySQL (XAMPP)

## Project Structure

* Frontend connected with Flask backend
* REST APIs for workers and bookings
* MySQL database for data storage

## APIs

* GET /api/workers
* POST /api/register_worker
* POST /api/book_service
* GET /api/bookings

## Installation

1. Clone the repository
2. Install dependencies
3. Start XAMPP MySQL server
4. Run Flask backend
5. Open the project in browser
